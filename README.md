Тестовое задание от компании TECCOD

1. Страница со списком услуг (данные можно захардкодить или замокать).

2. Для каждой услуги показать:

- название
- цену
- кнопку «Добавить»

3. При нажатии на кнопку услуга добавляется в список выбранных.
4. Отобразить блок «Итого»:

- выбранные услуги
- общую сумму
- кнопку «Оформить заказ» (без логики оплаты)

5. Интерфейс должен быть адаптивным (mobile-first).

Технические требования:

- React + TypeScript
- функциональные компоненты
- корректная работа со состоянием
- аккуратная и понятная структура кода
